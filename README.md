# AI Career Mentor & Job Application Automation Platform

## Problem Statement

Students struggle with managing job applications, resume building, interview preparation, and career planning.

This platform combines Full Stack Development, Machine Learning, Generative AI, and Automation into one application.

---

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

ML:
- Python
- Scikit-learn

GenAI:
- OpenAI API

Automation:
- Node Cron
- Nodemailer

---

## Features

- Authentication
- Resume Upload
- Resume Analysis
- Job Tracker
- AI Career Mentor
- Mock Interviews
- Weekly Reports
